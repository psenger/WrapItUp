jQuery.fn.wrapItUp

wrapItUp is a jQuery pluggin designed to take the selected inner text of a jQuery Selection and for each character wrap it with a tag. By default space is converted to non-break space.

Example

$('#target').wrapItUp('foo');
$('#target').wrapItUp('foo','goo');
$('#target').wrapItUp({className:['foo','goo']});
$('#target').wrapItUp({
                 className:['foo'],
                 innerDecorator: function ( i, v ) {
                                      if (v === " ") {
                                          return " ";
                                      } else {
                                          return (v?v.toUpperCase():v);
                                     }
                                  }
                 });
